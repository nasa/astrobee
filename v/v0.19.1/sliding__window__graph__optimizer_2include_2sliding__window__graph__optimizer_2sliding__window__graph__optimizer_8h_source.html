33 // GraphOptimizer that performs sliding window optimization.
34 // Calculates the desired window size in duration and number of nodes for each sliding window node adder
35 // using the SlidingWindowNodeAdders in the graph.
36 // Removes old nodes outside of window size along with the factors that depend on them after optimization.
37 // Adds either prior factors using marginalized covariances for new start nodes or adds
38 // marginal factors consisting of linearized errors for each factor removed (set desired behavior using params).
39 // Add sliding window node adders using the AddSlidingWindowNodeAdder() function (instead of AddNodeAdder() function) to
40 // ensure SlideWindow() is called for the sliding window node adder.
41 class SlidingWindowGraphOptimizer : public graph_optimizer::GraphOptimizer {

55  void AddSlidingWindowNodeAdder(std::shared_ptr<node_adders::SlidingWindowNodeAdder> sliding_window_node_adder);
